# Sun

简洁的 Next.js 16 + Tailwind CSS 起始项目，用于快速开始业务开发。

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 项目结构

```
apps/sun/
├── app/             # Next.js 应用目录
│   ├── layout.tsx   # 根布局
│   ├── page.tsx     # 首页
│   └── globals.css  # 全局样式
├── public/          # 静态资源
└── package.json     # 项目配置
```

## 环境要求

- Node.js >= 18.0.0
- pnpm >= 9.6.0
